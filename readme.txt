Add Canada Post shipping rates to WooCommerce as a shipping option for your customers. 
== Description ==

Canada Post Shipping For WooCommerce is a plugin that adds support for Canada Post shipping to WooCommerce. 

After installing the plugin your customers will have the option to choose Canada Post as a shipping method 
during the checkout. The plugin automatically calculates the letter shipping rates for your customers by using 
the dimensions and weights you provide on each of your inventory items.

If you need help with the plugin please feel free to email at nalidixic@gmail.com.

More detailed instructions and help documents can be found on the 
[Canadian Shipping Calculator](http://www.canadianshippingcalculator.com/) website.

== Installation ==

1. Download the plugin and activate it
2. Go to WooCommerce > Settings > Shipping
3. Enable the Canada Post shipping method
